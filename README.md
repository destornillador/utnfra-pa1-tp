# TP Programacion AV I

## Requisitos

* JDK >= 1.8 (no fue probado con otras versiones)
* MySQL 5.7
* Docker
* Docker Compose

## Inicializar DB

```
$> docker-compose up -d
```

## Ejecutar aplicación

La clase [App.java](src/App.java) es la entrypoint de la aplicación. Debe ser
compilada y ejecutada.
